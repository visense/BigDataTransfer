# 大数据中心文件传输助手 v1.0.1

首个正式版本发布！

## 新特性

- 系统托盘集成，最小化到托盘
- 实时显示连接状态
- 开机自启动选项
- 中文界面支持
- 一键启动/停止服务

## 更新内容

- 安装程序优化：默认选中"创建桌面快捷方式"和"开机自启动"选项
- 提升用户体验：减少初始配置步骤

## 系统要求

- Windows 7 及以上版本
- .NET 6.0 运行时
- 管理员权限（用于服务管理）

## 安装说明

1. 下载安装程序 `大数据中心文件传输助手_Setup.exe`
2. 以管理员身份运行安装程序
3. 按照安装向导完成安装
4. 运行桌面快捷方式或开始菜单中的程序
